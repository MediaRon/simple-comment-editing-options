Simple Comment Editing options for your website.

== Description ==

Simple Comment Editing Options provides an options panel for Simple Comment Editing with a myriad of optios including adjusting the timer, selecting a theme, and edit logging.

Simple Comment Editing Options features:
<ol>
<li>Set the timer to a custom value and even hide it</li>
<li>Select a theme</li>
<li>Receive an email each time a user edits or deletes their comment</li>
<li>Set a minimum comment length when editing<li>
<li>Log edits so you can see an edit history when viewing the comment.</li>
<li>Edit button text and other messages.</li>
</ol>

== Installation ==

1. Just unzip and upload the "simple-comment-editor-options" folder to your '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
